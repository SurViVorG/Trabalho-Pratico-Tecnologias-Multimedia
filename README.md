Endless runner "reverso"
Em vez de ser o jogador a correr eternamente atrás de objetos para obter uma melhor pontuação, o jogador está confinado a uma pequena arena na qual 
vão surgir os obstáculos que o jogador vai ter que evitar. Com animação, câmara presa em terceira pessoa, música, sons de impacto e game over, 2 personagens
e os vários menus necessários com tabela de Highscores.

Controlos 
WASD - andar
ESC - abrir o menu pausa

Como abrir o projeto
Abrir Assets/Scene/MainMenu e começar o jogo

Foram utilizadas imagens jpg convertidas em sprite para os menus e para paredes e chão foram utilizadas texturas da loja da Unity em que foram alteradas as escalas
para alinharem se melhor com os modelos necessários

Todos os sons utilizados foram provenientes de fontes externas em formato mp3 para facilitar a sua compatibilidade

O jogo foi inicalmente pensado para ser corrido em 1920 x 1080 pelo que após alguns problemas é utilizada a resolução atual com stretch

As animações do kyle funcionam mas um problema impede a alteração de uma para a próxima usando states ou blend tree
O menu highscores quando é aberto após a finalização de uma partida não aparece
